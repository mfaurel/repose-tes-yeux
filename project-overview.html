👁️ Repose Tes Yeux — Complete Project Analysis

Project Overview

Repose Tes Yeux (French: "Rest Your Eyes") is a Windows tray application implementing the 20/20/20 rule:

Every 20 minutes, take a 20-second break to look at something 20 meters away.

Key Facts:

  • Language: C# (.NET 8, WinForms)
  • Platform: Windows 10/11 (64-bit)
  • No admin rights required (single portable .exe)
  • Version: 0.1.0
  • Test Framework: xUnit (45 automated tests)
  • Multi-screen support with adaptive overlays

Core Components

1. EyeTimer (State Machine)

Responsibilities: Main timing logic, state transitions, break scheduling

  • Uses injectable IClock for testability (system or fake clock)
  • States: Idle → Working → Break (or Paused)
  • Events: OnTick, OnBreakStart, OnBreakEnd, OnStateChanged, OnAutoSuspended, OnAutoResumed
  • Tracks: breaks today, work seconds accumulated, break count since long break

Key Features:

Work phase with configurable interval (1–120 min)
Break phase with configurable duration (5–300 sec)
Auto-suspend on inactivity (via IdleDetector)
Auto-suspend in presenter mode (via PresenterModeDetector)
Long breaks every N breaks (configurable)
End-of-day break after X hours accumulated
Do-Not-Disturb window (no breaks during HH:mm – HH:mm)

2. TrayController (UI Orchestrator)

Responsibilities: System tray icon, context menu, overlays, sounds

  • Subscribes to all EyeTimer events
  • Dynamic icon with countdown (text-rendered, anti-aliased)
  • Context menu: Pause/Resume, Break Now, Profiles, Settings, Stats, Quit
  • Spawns OverlayForm on each screen during break
  • Plays start/end sounds (customizable .wav or system beep)
  • Profiles submenu with switch, save-as, delete

3. OverlayForm (Break Notification)

Responsibilities: Full-screen, non-dismissible break reminder

  • Positioned bottom-right of each screen
  • TopMost window (always visible)
  • Shows: Rule label, message, instruction, progress bar, countdown, skip button (if enabled)
  • Adaptive background (tinted desktop wallpaper if enabled)
  • Auto-updates every 1000 ms

4. SettingsForm (Configuration Panel)

Responsibilities: All user-configurable settings

  • Work interval, break duration, distance recommendation
  • Custom overlay message & language selection
  • Startup hook, overlay dismiss button, sound settings
  • Do-Not-Disturb time window
  • Long break settings (frequency, duration, message)
  • End-of-day reminder settings
  • Inactivity detection & presenter mode suspension
  • Adaptive overlay toggle
  • Persists changes and updates timer in real time

5. AppSettings (Typed Configuration)

Responsibilities: Configuration model with validation

  • 31 typed properties (int, bool, string, enum)
  • Default values for all settings
  • WithDefaults() method clamps values to safe ranges
  • Computed properties: HasDoNotDisturbWindow, DoNotDisturbWindow
  • JSON-serializable (System.Text.Json)

6. SettingsStore & ProfileStore (Persistence)

SettingsStore: Single settings.json in %APPDATA%\ReposeTesYeux\

  • Atomic save with error recovery
  • Graceful degradation (missing file = defaults)

ProfileStore: Named profiles in profiles.json

  • Switch profiles via tray menu
  • Save current config as new profile
  • Delete non-default profiles
  • Ensures "Default" profile always exists

7. BreakHistory (Statistics)

Responsibilities: Break counting and history

  • Persists daily break counts (history.json)
  • Provides: today, 7-day, 30-day totals
  • Trims data older than 35 days automatically
  • Used by StatsForm to render 28-day bar chart

Architecture Patterns

Dependency Injection: Constructor-based DI for EyeTimer (IClock)

Event-Driven: EyeTimer publishes events; TrayController subscribes

State Machine: EyeTimer manages 4 states with explicit transitions

Isolation: Settings, History, Profiles stored independently in %APPDATA%

Testability: FakeClock allows time control in unit tests

Error Handling: Graceful degradation (default values on parse/load failures)

Win32 Interop: P/Invoke for tray icon, idle detection, presenter mode, wallpaper
